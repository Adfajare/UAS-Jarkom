# Awas Bom
## Guessing the right number

A Java-based network game where players guess the correct number to avoid the bomb!

### How to Build
```bash
javac server/*.java client/*.java
```

### How to Run

#### Start the Server
```bash
cd server
java -cp . GameServer
```

#### Start the Client
```bash
cd client
java -cp . GameClient
```
